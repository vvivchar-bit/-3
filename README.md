## Короткий опис

У межах практичного заняття було опрацьовано тему поліморфізму, віртуальних функцій та абстрактних класів.

У репозиторії подано дві програми:

1. **ConsoleApplication1** — демонстрація використання абстрактного класу `Figure` і похідних класів `Triangle`, `Rectangle`, `Circle` для обчислення площ фігур.
2. **ConsoleApplication2** — демонстрація використання абстрактного класу `Convert` і похідних класів для перетворення величин.
